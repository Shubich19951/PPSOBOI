## Схема работы приложения
На выделенном сервере Amazon запущен RabbitMQ, принимающий на вход текстовые записи - логи приложений  
Поступающие сообщения сохраняются в MongoDB  

Затем Hadoop берет исходные данные, делает пост-обработку и сохраняет результат в БД  

### Посылаем сообщения в RabbitMQ  
```python new_task.py --message=TEXT```  

### Разгружаем очередь и сохраняем результаты  
```python worker.py --mongo_uri=""```
